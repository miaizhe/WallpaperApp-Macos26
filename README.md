# Liquid Wall (液态壁纸)

Liquid Wall 是一款专为 macOS 设计的现代化动态壁纸应用。它采用 SwiftUI 开发，融合了精美的液态玻璃（Glassmorphism）UI 设计风格，为您带来流畅、沉浸式的桌面体验。

## ✨ 主要特性

*   **🎬 动态视频壁纸**: 支持导入 `.mp4`, `.mov` 等格式的视频文件作为桌面壁纸，让您的桌面动起来。
*   **💧 液态玻璃界面**: 采用半透明磨砂与流体渐变动画设计，界面美观且极具现代感。
*   **🖥️ 多显示器支持**: 自动识别并适配所有连接的显示器，为每个屏幕独立渲染壁纸。
*   **🚀 极致性能**:
    *   原生支持 **Apple Silicon (M1/M2/M3)** 和 **Intel** 芯片（通用二进制）。
    *   使用 `AVPlayerLooper` 实现视频无缝循环播放，无黑屏闪烁。
    *   智能资源管理，窗口关闭时自动释放资源。
*   **🎛️ 状态栏控制**:
    *   快捷播放/暂停。
    *   一键切换“下一张”或“随机”壁纸。
    *   支持直接从菜单栏列表中选择特定壁纸。
*   **📚 壁纸库管理**:
    *   自动生成视频缩略图预览。
    *   保存播放历史，方便快速切换。
    *   支持拖拽窗口（通过特殊处理的背景区域）。
*   **🎨 个性化设置**:
    *   **透明度调节**: 自定义应用界面的卡片透明度。
    *   **背景定制**: 支持设置在线 API（如 Picsum）作为应用背景图。
    *   **音量控制**: 独立控制壁纸视频的音量。
*   **🌐 多语言支持**: 内置简体中文和英语，根据系统语言自动切换。

## 📦 安装说明

### 方法一：直接安装 (推荐)
1.  下载最新的 `DynamicWallpaperApp.dmg` 安装包。
2.  双击打开 DMG 文件。
3.  将 **DynamicWallpaperApp** 拖入 **Applications** (应用程序) 文件夹。
4.  从启动台或应用程序文件夹运行即可。

### 方法二：从源码构建
如果您是开发者，可以手动编译项目：

**环境要求**:
*   macOS 12.0 或更高版本
*   Swift 5.9+ / Xcode 15+

**构建步骤**:
1.  克隆代码库到本地。
2.  在终端中进入项目目录。
3.  运行打包脚本（自动构建通用二进制并生成 DMG）：
    ```bash
    chmod +x package_dmg.sh
    ./package_dmg.sh
    ```
4.  或者直接运行调试：
    ```bash
    swift run
    ```

## 📖 使用指南

1.  **导入壁纸**: 点击主界面上的 "Import" (导入) 按钮或 "+" 号，选择本地视频文件。
2.  **切换壁纸**: 在右侧 "Library" (壁纸库) 中点击任意卡片即可切换。
3.  **状态栏菜单**: 点击顶部菜单栏的 "✨" 图标，可进行快速控制。
4.  **设置**: 点击左侧边栏底部的 "Settings" (设置) 按钮，可调整透明度和背景源。
